Personal website template

You can add an image in the same folder as this HTML page, if you call it background.jpg it should automatically cover the screen as a background image.

This page also has a white transparent background which allows you to see the background image without making the text too difficult to read. Change the settings in the style.css page to match it to your background.

If you look at the HTML document you will see that you can also add a picture of yourself that will show up on the right-hand side of the page with all of this text to the left.
